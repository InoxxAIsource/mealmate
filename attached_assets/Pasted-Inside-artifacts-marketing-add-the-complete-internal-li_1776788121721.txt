Create artifacts/marketing/components/seo/
RelatedLinks.tsx:
Props: { currentPath: string }
Looks up linkGraph[currentPath] and renders as 
a styled 'Related Pages' grid of cards.
Each card: title, descriptive anchor text, arrow →
Never use generic anchor text like 'click here' 
or 'read more'.
Add to bottom of every condition page, 
programmatic page, and blog post.

Create artifacts/marketing/components/seo/
Breadcrumbs.tsx:
Props: { items: Array<{label, href}> }
Renders breadcrumb trail with JSON-LD 
BreadcrumbList schema.
Format: Home > [Section] > [Page]
Each crumb is a link except the last.
Add to all condition pages, programmatic pages, 
blog posts, FAQ, About.

INTERNAL LINK RULES — apply to every page:
1. Every page links to at least 4 other pages 
   with descriptive keyword-rich anchor text
2. No page is more than 3 clicks from homepage
3. Every blog post links to its parent condition 
   page (e.g. PCOS blog post → /pcos-meal-plan)
4. Every programmatic page links to its parent 
   condition page
5. Every page ends with a CTA linking to 
   app.mealcoreai.com/sign-up with 
   condition-specific anchor text
6. Condition pages cross-link to each other 
   where medically related (PCOS ↔ Diabetes, 
   Pregnancy ↔ Kids, Thyroid ↔ Cholesterol)
